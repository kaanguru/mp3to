# mp3to

mp3to converts mp3 audio files to other formats such as , ogg (default), flac and AAC.

It requires ffmpeg to be installed on the user's system.

# Usage

<!-- usage -->
```sh-session
$ npm install -g mp3to
$ mp3to COMMAND
running command...
$ mp3to (--version)
mp3to/1.0.0 win32-x64 node-v22.21.0
$ mp3to --help [COMMAND]
USAGE
  $ mp3to COMMAND
...
```
